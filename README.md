Fair Share is a product case study and clickable prototype for a household fairness app. Most expense-splitting tools (like Splitwise) only track money — Fair Share combines shared expenses and chores into one fairness score, so labor imbalance doesn't stay invisible. Includes the full PO process: problem research, competitive analysis, RICE prioritization, user stories, and a 5-screen interactive prototype.
